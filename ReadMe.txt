仓库里整理11级从11年入学到18年毕业这几年里所保存下来的课件、笔记参考资料等，供大家学习参考。

毕业前总是在抱怨学了一堆没啥卵用的东西，感觉学的杂而不精，所以很多课都没有好好上。上班之后才发现之前真是too young too simple了，学的东西是真的有用（尤其是预科的语言和数理化，是真心有用），只可惜自己没有好好学，现在是追悔莫及。上边这些话主要是写给那些没有毕业的同学们的，你们还有机会，要珍惜大学的时间，尽管可能你们觉得学院有这样和那样的毛病，但是抱怨是没用的，有时间抱怨还不如抓紧时间充实一下自己，7年的时间会影响你今后的人生，值得一拼，不过我知道我说这些话多半也没啥卵用，包括我在内的大多数人不吃亏是不长记性的，等真吃了亏也已经错过了，所以随缘吧。

对于已经毕业的同志们，这个仓库的价值主要在于建立一个知识储备库，帮助大家在工作中重新拾起相应的知识，希望可以对大家的工作有所助力。
